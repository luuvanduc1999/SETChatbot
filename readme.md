## Install Rasa + Các module khác
- > pip3 install -r requirements.txt i

## Train model
- > rasa train

hoặc chỉ train nlu:
- > rasa train nlu

## Run
**Nếu có custom action cần chạy song song 2 terminal**
- > rasa run actions
- > rasa shell

**Run API**
- > rasa run actions
- > rasa run --enable-api
